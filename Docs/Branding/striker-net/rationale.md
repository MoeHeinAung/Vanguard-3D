# Striker.Net — Branding Rationale

> **Tagline Options:** BETTING INTELLIGENCE · NETWORK COMMAND

## Name Breakdown

- **STRIKER** — A sharp, aggressive name implying accuracy, calibration, and decisive action. In the betting context it reads as "goal scorer," "winner," or "lucky strike" — all positive associations with draw outcomes and agent success.
- **.NET** — A double-layered suffix: it references both the `.net` top-level domain *and* the "network" topology at the heart of the product (agents → master dealers → head office). It is instantly readable to non-English speakers worldwide.
- **STRIKER.NET** — Reads as both a technology domain *and* a stylised lowercase "S" formed inside the mark (the crimson diagonal slash through the shield/diamond shape).

The name also preserves a natural pronouncable short name — *Striker* — which works well for product roadmaps (Striker CLI, Striker SDK, Striker Cloud).

## Visual Design Choices

### Colour Palette
| Role | Hex | Rationale |
|---|---|---|
| Background | `#070b1a` | Very deep navy-black; darkest of the three options — feels premium and authoritative |
| Primary Violet | `#a78bfa` | Royal violet — signals intelligence and strategy over brute force |
| Magenta | `#7c3aed` | Deep purple accent; supports node differentiation visually |
| Cyan Teal | `#22d3ee` / `#10b981` | Success/positive — ties to settlement outcomes and winning tickets |
| White text | `#ffffff` | Standard; bright on the darkest background |

### Network Topology Mark
The mark is a **connected-node graph** — 6 satellite nodes radiating from a central hub using faint edge lines. The central icon is a **shield pushed onto its point**, hinting at:
- **Strike** — the sharp diagonal charcoal bar inside evokes a hit, a win, a correct-prediction.
- **Node** — a basic interconnection unit, representing any node type (agent, MD, draw, sales, risk, settlement).
Each satellite is colour-coded by type:
| Node | Colour | Label |
|---|---|---|
| Agent | `#22d3ee` cyan | A |
| Master Dealer | `#10b981` emerald | MD |
| Draw | `#a78bfa` violet | D |
| Sales | `#f59e0b` amber | S |
| Risk | `#34d399` green | R |
| Settlement | `#818cf8` indigo | X |

### Wordmark
- **STRIKER** — Bold white; clean, authoritative, stands for 100 modules and CLI users.
- **.NET** — violet; smaller and set negative (softer than the primary weight), allowing the ".NET" suffix to float slightly above the baseline and feels integrated.

### Supporting Elements
- Dot-mesh background cluster is a semi-transparent grid suggesting network latency/topography and readouts.
- The dot-grid strip at the bottom (echoed from the other logos) anchors brand consistency across the suite.
- The dark background is the warmest of all three, making Striker.Net the most "premium console" feel.

## When to Use Striker.Net

- Rebranding around the network graph / topology theme — agent-dealer referral trees, master dealer liability rings.
- Releasing an API-first / SDK-first product where `.NET` is a persuasive and logical domain name.
- Positioning the product as competitive intelligence or risk-prediction software.
- International markets where "Striker" (strike = hit, score) translates to good-luck / action in multiple languages.
